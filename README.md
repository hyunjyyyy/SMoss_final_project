# 오픈소스프로그래밍 기말 프로젝트

## 프로젝트 개요
이 프로젝트는 재활용품 이미지 분류를 수행합니다. 다양한 종류의 재활용품들을 각각의 카테고리로 분류하고, 그 재활용품이 무엇인지 알려주는 TTS모델까지 포함합니다.
